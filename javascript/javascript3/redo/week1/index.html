Movies API

This API can be find in the link: "https://developers.themoviedb.org/3/getting-started/introduction".
It is a JSON data API which gives list of movies and their reviews which we can analyse
the movies comments and reviews with natural language processing and give a list of recommended movies based on them.
This API is an array of objects which includes title, id, release_date, overview, ... which can be used for processing.
We can access to this API by a key which their website gives us.

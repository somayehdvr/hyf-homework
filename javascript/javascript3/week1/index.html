Stock News API

This API can be find in their website: "stocknewsapi.com".
It is a JSON data API which gives an up to date list of stock news and the changes in the stocks which will help us to find a good stock to invest in.
Also with use of this API we can develop applications to help the investors to decide better in their investment.
This API is an array of objects which includes news url, image url, title, text, date, topic, sentiment, ticker, type, and source name.
We can access to this API by a key which their website gives us.
